NodeText

Lightweight Code Editor with Rust Debugging

A fast, free code editor built with Rust and Tauri featuring integrated LLDB debugging, syntax highlighting for 20+ languages, and a built-in terminal. Designed for developers who want a lightweight yet capable editor with first-class Rust support.

Key Features

Integrated Rust Debugging

Full LLDB debugger integration with breakpoints (click the gutter to set), conditional expressions, thread ID filtering, and step controls (Step In, Step Over, Step Out, Continue, Pause). Supports MSVC PDB on Windows.

Variable Inspection

Watch and Locals panels for inspecting local variables, adding watch expressions, and viewing String contents, and Vec lengths during debugging sessions.

Syntax Highlighting

Viewport-based syntax highlighting for 20+ languages with zero lag on large files. Auto language detection from file extensions with accurate tokenization for all supported languages.

Built-in Terminal

Integrated shell panel (Ctrl+`) for running cargo, git, and other commands directly within the editor. Full ANSI color support for rich terminal output.

Code Navigation

Tree-sitter powered symbol search for Rust code with find definition and find references commands. Navigate your codebase efficiently with AST-aware code intelligence.

Cargo Integration

Built-in cargo check with JSON diagnostics parsing for real-time error and warning reporting. Run cargo build directly from the editor with structured output.

Find & Replace

Ctrl+F for find with real-time match highlighting (orange for current match, blue for others). Ctrl+H for find and replace with Replace All. Project-wide search with Ctrl+Shift+F.

Auto LLVM Install

Automatically downloads and installs LLVM with a progress dialog if no debugger is found on the system. Zero-friction setup for Rust debugging on Windows.
